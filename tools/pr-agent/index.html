        내부 테스트 발송 체크리스트
        ⚑ 실제 기자 발송은 한 번에 전체가 아니라 3~5명 파일럿으로 먼저 진행한 뒤, 수신율·스팸 여부를 확인하고 점진적으로 확대합니다.
        STEP 6 · 발송 준비 & 발송

        발송 준비

        발송 모드를 선택하세요. 기본값은 내부 테스트 발송(TEST_EMAIL)이며, 기자 파일럿 발송은 내부 검수 완료 후에만 활성화됩니다.

        🔍 1차 내부 검수 (기자 발송 전)
        현재 단계는 기자 발송 전 내부 검수 단계입니다. 기자에게 바로 보내지 않고, TEST_EMAIL로 먼저 발송 품질을 확인하고 임시값·누락 정보·위험요소를 점검합니다.
        필수 체크리스트 (0/8)
        ※ 기자 발송은 4개 조건(임시값 0 · TEST_EMAIL 발송 성공 · 보도자료 80점↑ · 체크리스트 완료)을 모두 만족한 뒤, 1명 파일럿부터 시작해 수신율·스팸을 확인하며 점진 확대합니다.
        ⚠️ 기자 파일럿 발송은 실제 기자에게 메일이 발송됩니다. 반드시 보도자료와 첨부자료를 검토한 뒤 1~5명 이내로만 발송하세요.
        파일럿 수신 기자 선택 (0/5)
        📎 첨부자료 현장·제품 사진 / 보도자료 PDF / 관련 자료 ZIP
        jpg · png · pdf · zip · 1개 10MB · 전체 25MB
