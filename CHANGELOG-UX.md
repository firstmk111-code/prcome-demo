## [공통 기반] 디자인 시스템 폴리시 — `assets/css/magazine-system.css`
전 페이지에 자동 적용되는 토큰·접근성·모션 레이어를 신설.

| 항목 | 개선 내용 |
|---|---|
| **컬러** | 텍스트 선택색(`::selection`)을 브랜드 블루로 통일 |
| **엘리베이션** | 그림자 토큰 3단계 신설(`--shadow-sm/md/lg`) — 임의 그림자 값 제거, 일관된 깊이 |
| **모션** | 이징·지속시간 토큰(`--ease`, `--ease-out`, `--dur`, `--dur-2`)으로 전환 통일 |
| **Accessibility** | 키보드 포커스 링(`:focus-visible`) 전역 적용 — 마우스 클릭 시엔 노출 안 함 |
| **Accessibility** | `prefers-reduced-motion: reduce` 존중 — 모션 최소화 사용자에겐 애니메이션/부드러운 스크롤 비활성 |
| **Animation** | 스크롤 진입 리빌 기반(`.reveal-js` 게이트) — JS 미동작 시 콘텐츠 항상 노출(안전 폴백) |

## [1] 홈 `index.html`

### 정보 밀도 / 여백
- 모바일(≤600px) 히어로·섹션 패딩 축소로 첫 화면 정보 밀도 개선.
- 초소형(≤400px) 도서 그리드 2열로 재배치(기존 3열 과밀 해소).

### 카드 디자인 / 컬러 / 엘리베이션
- 뉴스 카드 그림자를 토큰(`--shadow-md`)으로 교체 — 더 부드럽고 일관된 깊이.
- 사이드 박스·도서 카드에 hover 시 미세 그림자/리프트 추가로 입체감 강화.

### Hover
- **대칭 전환 버그 수정**: 뉴스 카드의 `transition`이 `:hover`에만 있어 마우스를 뗄 때 즉시 튀던 문제를 base로 이동해 **들어올 때·나갈 때 모두 부드럽게**.
- 뉴스 카드 hover 시 썸네일 미세 확대(scale 1.04)로 상호작용 피드백 강화.
- 커뮤니티/매거진/디렉토리 행 hover에 은은한 좌측 그라데이션 하이라이트.
- 모든 색 변화(제목·링크·칩)에 부드러운 색 전환 적용(기존 즉시 변경).
- 주요 버튼(히어로 검색·AI 밴드·가입) hover 시 미세 리프트 + 그림자.

### Animation
- 섹션 단위 **스크롤 진입 리빌**(fade + up) 적용 — IntersectionObserver 기반.
- 콘텐츠가 숨은 채 남지 않도록 **1.8s failsafe 타이머** 추가(관측 실패 시 강제 노출).
- `prefers-reduced-motion` 사용자에겐 리빌 비활성(콘텐츠 즉시 표시).

### Loading
- 뉴스 썸네일·도서 표지 이미지 `loading="lazy"` 확인/유지 — 초기 로드 부담·CLS 완화.

### Empty State
- 커뮤니티 피드 빈 상태를 **일러스트 아이콘 + 안내 문구 + ‘글쓰기’ CTA** 카드로 개선(기존 단순 텍스트 “아직 글이 없습니다”).

### Responsive
- ≤600px: 히어로/섹션 여백 축소, 퀵내비 좌측 정렬.
- ≤400px: 도서 그리드 2열.

### Accessibility
- 전역 `:focus-visible` 포커스 링 적용(키보드 탐색 가시성).
- 모션 최소화 선호 존중.

## [2] Company Detail `directory/company.html` — 전면 리디자인 (SaaS 수준)

> Sprint 목표: 정보 계층 · 카드 디자인 · CTA · 통계 · 신뢰 요소 · 반응형 · Skeleton/Empty · Hover 애니메이션.

### 정보 계층 개선
- 기존 단일 컬럼 나열 → **히어로 + 통계 스트립 + 2열(본문 / 스티키 사이드바)** 구조로 재편.
- 히어로: 로고·회사명(H1)·한 줄 소개·배지·핵심 CTA를 최상단에 집약.
- 사이드바에 **회사 정보 카드**와 **CTA 카드**를 고정(sticky)해 스크롤 내내 행동 유도.

### 카드 디자인 개선
- 라운드 16px·토큰 그림자 기반의 카드 시스템으로 통일.
- 제공 서비스를 **아이콘 카드 4종 그리드**로(기존 불릿 리스트).
- 관련 회사를 **로고 카드 3종**으로.

### CTA 추가
- 히어로(공식 홈페이지 / 홍보대행 문의) + 사이드 CTA 카드(의뢰 / 단가표) + 하단 CTA 밴드 + 공식 홈페이지 밴드 — **총 7개 행동 경로**.

### 통계 영역 추가
- **4개 스탯 타일**: 구분(국내/글로벌) · 업종(PR) · 서비스 영역(4+) · 비교 가능 회사 수(66곳).

### 신뢰 요소 추가
- **한국PR협회 등록** 인증 배지(히어로).
- **신뢰 정보 카드**: 협회 등록·정보 표기·공식 채널 연결 체크 항목 + 출처 링크·참고용 고지.

### Skeleton / Empty State
- **Skeleton**: JS 렌더 전/미동작 시 히어로·통계 자리에 shimmer 플레이스홀더(모션 최소화 시 정지).
- **Empty State(Not Found)**: 잘못된 `?c` 접근 시 조용한 리다이렉트 대신 **‘회사를 찾을 수 없어요’ 카드 + 디렉토리 CTA**.

### Hover 애니메이션
- 스탯·서비스·관련·정보 카드 hover 시 리프트 + 토큰 그림자.
- 버튼 hover 시 미세 리프트 + 화살표 슬라이드(→).

### 반응형 개선
- ≤900px: 2열 → 1열, 사이드바가 2열 그리드로 하강, 통계 2열.
- ≤640px: 서비스·관련·사이드·통계 단열, 히어로 로고 축소.

### Accessibility / Animation
- 섹션 단위 스크롤 리빌(+1.8s failsafe) · `prefers-reduced-motion` 존중 · 공통 `:focus-visible` 상속.
- 외부 홈페이지 링크 `nofollow` 유지.
